## 📌 Selected Projects

### 🛡️ Intelligent Insurance Claims Assistant
> **Multi-Agent System** · AWS Bedrock · Emotion Detection · Data Security

Built an enterprise multi-agent AI system for insurance companies that reads policy documents, claims procedures, and workflows to provide accurate answers to customer inquiries. Integrated emotion detection to identify customer distress and trigger human intervention. Implemented secure data transmission using AWS Bedrock Gateway.

### 🔍 Enterprise RAG Document Analysis System
> **RAG Pipeline** · LangChain · Pinecone · LlamaIndex

Designed and deployed an enterprise-grade RAG system that analyzes multi-format documents (PDFs, text) containing professional domain knowledge. Achieved **95% extraction accuracy** across 20+ customer profiles with fine-tuned transformer-based LLMs.

### 🎙️ Multi-Modal Emotion Classification Engine
> **Audio AI** · Wave2Vec2 · Qwen · GPT-4 Annotation

Developed multi-modal foundation models combining speech and language models for emotional classification of 10K+ audio samples, achieving **90% accuracy**. Designed automated annotation workflows using LLM APIs, reducing manual labeling time by 80%.

### ⚡ Real-Time Audio Analysis Platform
> **Full-Stack** · FastAPI · Swift iOS · PostgreSQL · AWS

Built an end-to-end real-time audio analysis application with Swift frontend and FastAPI backend, achieving **sub-100ms response latency**. Deployed with Docker containerization and AWS cloud infrastructure for production workloads.
